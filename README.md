# 🍜 ラーメンタイマー with ミクダヨー

> 「のびないように見張ってるよ。」 - ミクダヨー

## 📖 概要

インスタントラーメンを完璧に茹で上げるための、世界で最も愛らしいタイマーアプリダヨー。

ミクダヨーが3分間、あなたのラーメンを優しく見守るダヨー。フタが開けられないように、のびないように、そして何より**完璧な茹で加減**になるようになるんダヨー。

## 🔗 デモ

このアプリは GitHub Pages にデプロイされています。ブラウザで以下の URL を開けば、すぐに遊べます:

https://pucho3939.github.io/ramen_timer_mikudayo/

## ✨ 特徴

- 🎯 **3分間の完璧なタイミング**: 180秒のカウントダウンで、あなたのラーメンを理想的な状態にするんダヨー
- 💬 **ミクダヨーの励ましの言葉**: 5秒ごとにランダムで表示される、心温まるメッセージダヨー
- 🎨 **美しいUI**: グラデーション背景とポップなアニメーションで、ラーメンタイムを楽しくするんダヨー
- 📱 **シンプル**: HTMLファイル1つで完結ダヨー。ブラウザで開くだけダヨー！

## 🚀 使い方

1. `index.html` をブラウザで開く
2. 「スタート」ボタンをクリック
3. ミクダヨーと一緒に3分間待つ
4. 🍜 できあがり！

**重要**: このタイマーはラーメンを茹でるわけじゃないヨー。実際にラーメンを茹でるのはあなたの仕事ダヨー。

## ⚠️ 注意事項

**ネット接続について**: このアプリはGood Smile Company社の画像と、Google Fontsを使用しているため、インターネット接続がない環境では表示が乱れる可能性があるヨー。オフライン環境でも動作はするけど、フォントがフォールバック（システムフォント）になるから、見た目が少し変わるかもしれないヨー。

## 🎭 ミクダヨーのセリフ集

タイマー中、ミクダヨーは以下のような励ましの言葉をかけてくれるんダヨー(初期のセリフだから今は違うヨー)：

- 「いい感じに待っていこう。」
- 「そろそろいい匂いしてきた？」
- 「フタ、ちゃんと閉めてる？」
- 「今は我慢の時間だよ。」
- 「湯気に注意してね。」
- 「のびないように見張ってるよ。」

## 📦 インストール

```bash
# このリポジトリをクローン
git clone https://github.com/yourusername/ramen_timer_mikudayo.git

# ディレクトリに移動
cd ramen_timer_mikudayo

# index.htmlをブラウザで開く
# Windowsの場合:
start index.html
# Macの場合:
open index.html
# Linuxの場合:
xdg-open index.html
```
